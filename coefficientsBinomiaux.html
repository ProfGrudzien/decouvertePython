Calcul de coefficients binomiaux

En mathématiques, le coefficient binomial (nk) donne (entre autre) le nombre de façon de réunir k éléments dans un ensemble de n éléments. Par exemples, (115) donne le nombre d'équipes de 5 joueurs choisis dans une classe de 11 élèves.

Les coefficients binomiaux peuvent se calculer par récurrence grâce aux relations :

  • (nk) = 0, si k > n
  • (nn) = 1, quelque-soit la valeur de n
  • (n0) = 1, quelque-soit la valeur de n
  • (nk) = (n - 1k - 1) + (n - 1k), quelque-soit les valeurs de n et de k

Vous devez écrire un programme récursif coefficient_binomiale(n, k) qui donne la valeur du coefficient binomial (nk).

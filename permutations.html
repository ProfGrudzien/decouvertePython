Liste des permutations

L'ensemble des permutations d'une liste est l'ensemble des listes composées des mêmes éléments que la liste de départ mais dans un ordre éventuellement différent. Par exemples l'ensemble des permutations de la liste L = [1, 2, 3, 4] est [[1, 2, 3, 4], [1, 2, 4, 3], [1, 3, 2, 4], [1, 3, 4, 2], [1, 4, 2, 3], [1, 4, 3, 2], [2, 1, 3, 4], [2, 1, 4, 3], [2, 3, 1, 4], [2, 3, 4, 1], [2, 4, 1, 3], [2, 4, 3, 1], [3, 1, 2, 4], [3, 1, 4, 2], [3, 2, 1, 4], [3, 2, 4, 1], [3, 4, 1, 2], [3, 4, 2, 1], [4, 1, 2, 3], [4, 1, 3, 2], [4, 2, 1, 3], [4, 2, 3, 1], [4, 3, 1, 2], [4, 3, 2, 1]]

Vous devez écrire de manière récursive, un programme donnant l'ensemble des permutations d'une liste L.

Vous utiliserez ensuite cette fonction, l'algorithme de recherche dichonotmique, ainsi que le fichier mots.francais.txt contenant plus de 300 000 mots de la langue française pour écrire une fonction donnant les anagrammes existants d'un mot en français.
